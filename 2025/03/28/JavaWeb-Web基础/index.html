JavaWeb-Web基础

JavaWeb
Publish Date:   2025-03-28

1. JavaWeb 概述

JavaWeb 主要用于开发基于 Java 的 Web 应用，核心技术包括 Servlet、JSP、过滤器、监听器、MVC 模式 等。

2. HTTP 及 Web 交互

  • HTTP 请求方式

    • GET：获取资源，参数暴露在 URL 中。
    • POST：提交数据，数据在请求体中，不会暴露在 URL。
    • PUT：更新资源。
    • DELETE：删除资源。

  • 常见状态码

    • 200 OK：请求成功。
    • 302 Found：重定向。
    • 404 Not Found：资源未找到。
    • 500 Internal Server Error：服务器内部错误。

3. Servlet 基础

Servlet 是 JavaWeb 的核心，用于处理 HTTP 请求和响应。

3.1 Servlet 生命周期

  1. 初始化：容器加载 Servlet 并调用 init() 方法。
  2. 请求处理：每个请求都会触发 service() 方法，根据请求类型调用 doGet() 或 doPost()。
  3. 销毁：当容器销毁 Servlet 时，会调用 destroy() 方法。

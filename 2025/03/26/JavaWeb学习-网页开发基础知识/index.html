HTML, CSS, JavaScript 笔记

本笔记涵盖了 HTML、CSS 和 JavaScript 的基础知识，帮助了解网页的构建要素及其相互关系。

1. HTML 基础

1.1 HTML 简介

HTML（HyperText Markup Language）是用于创建网页和 Web 应用程序的标准标记语言。它通过标签来描述网页的结构和内容。

1.2 常见 HTML 标签

  • <html>：文档的根元素
  • <head>：包含元数据、样式和脚本等信息
  • <title>：网页标题
  • <body>：网页主要内容
  • <h1> ~ <h6>：标题标签
  • <p>：段落
  • <a>：超链接
  • <img>：图像
  • <ul>, <ol>, <li>：无序列表、有序列表和列表项
  • <div>：区块级容器
  • <span>：行内容器

2. CSS 基础

2.1 CSS 简介

CSS（Cascading Style Sheets）用于描述 HTML 文档的外观和布局。它控制网页的颜色、字体、间距、排版等视觉效果。

2.2 CSS 选择器

标签选择器：直接选择 HTML 标签，例如 p, h1。

类选择器：使用 . 开头，例如 .className。

ID 选择器：使用 # 开头，例如 #idName。

组合选择器：例如后代选择器、子选择器、兄弟选择器等。

3. JavaScript 基础

JavaScript 是一种解释型编程语言，主要用于为网页增加动态交互功能。它可以操作 HTML 和 CSS，改变网页内容、样式和结构。

3.1 常见语法与概念

变量声明：使用 var、let 或 const 声明变量。

数据类型：包括字符串、数字、布尔值、数组、对象等。

函数：封装可复用的代码块。

条件语句：例如 if、else。

循环语句：例如 for、while。

JavaScript 添加动态功能，操作 HTML 和 CSS。
